How to Cite Fast C4.5

To cite our work, please cite the following document:

Ping He, Ling Chen and Xiaohua Xu, Fast C4.5, International Conference on Machine Learning and Cybernetics, 2007, vol. 5 : pp. 2841-1846.

The bibtex format is

@inproceedings{HLP07,
   author="Ping He and Ling Chen and Xiaohua Xu",
   title="Fast C4.5",
   booktitle="Proceedings of the International Conference on Machine Learning and Cybernetics 2007",
   year="2007",
   note="Software available at http://code.google.com/p/fastc45/downloads/list"
}
